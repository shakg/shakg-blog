# Ishak's Thoughts

A minimalist blog built with [Astro](https://astro.build).

## About

A personal blog where I write about software development, systems design, and the occasional deep dive into something unexpected.

## Tech Stack

- **Framework**: Astro 4
- **Icons**: Lucide
- **Language**: TypeScript

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

The blog is optimized for deployment on Vercel or Netlify.
